# BookStore (cleaned)

This is a cleaned Spring Boot 3.x BookStore example project using H2 and JWT authentication.

### Run

1. mvn clean spring-boot:run
2. H2 console: http://localhost:8080/h2-console (JDBC URL: jdbc:h2:mem:bookdb)

### Notes
- Move JWT secret to environment variable in production.
- This scaffold contains essential controllers/services/repositories. Add more fields and mapping as needed.
